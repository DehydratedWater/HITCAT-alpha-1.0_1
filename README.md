# HITCAT-alpha-1.0_1

Sterowanie:
spacja - przełączanie pomiędzy trybem śledzenia kota, a wolnej kamery
w, a, s, d - kierowanie ciałem kota 
backspace - unieruchomienie kota
scroll - zoomowanie
strzałki - kierowanie kamerą (tylko przy wyłączonym trybie śledzenia kota)
nawiasy kwadratowe - zwiększanie i zmniejszanie szybkości ruchu kota lub kamery
